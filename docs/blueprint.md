# **App Name**: UroTrack

## Core Features:

- Urinary Symptom Tracker: Collect patient-reported urinary symptoms, including urgency and burning sensations during urination, post-radical prostatectomy.
- Erectile Function Tracker: Track erectile function, including medication usage (Tadalafil 5mg, Tadalafil 20mg, Sildenafil) and associated outcomes.
- PSA Result Logger: Log PSA test results and relevant post-operative data for longitudinal tracking.
- Dashboard Visualization: Generate a dashboard compiling all user-provided data into interactive, informative visuals, making trends easily identifiable.
- Secure Login: Enable secure, password-protected access to ensure data privacy and user authentication.
- Data Persistence: Provide data persistence so data inputted will persist through interactions of the app, and page refreshes. 
- Referral Link Generator: Provide a link and suggested text that patients can copy and paste in their messaging app of choice in order to share the UroTrack App. A generative AI tool will intelligently shorten the provided URL to use the least characters possible.  

## Style Guidelines:

- Primary color: Muted blue (#77B5FE) for a sense of trust and clinical precision.
- Background color: Very light blue (#F0F8FF), almost white, creating a clean and professional feel.
- Accent color: Soft lavender (#E6E6FA) for interactive elements and highlights, offering a gentle contrast.
- Headline font: 'Inter', a modern sans-serif for a clean and objective look.
- Body font: 'PT Sans', a humanist sans-serif providing a little warmth and personality, yet suitable for longer text.
- Use clear, minimalist icons to represent different symptoms and data points for easy recognition.
- Implement a responsive, mobile-first design to ensure accessibility and ease of use on all devices.
